# UNIT-1-JAVA-PROJECT

This repository contains **Unit–1 Java console-based applications** developed as part of academic coursework.  
The programs focus on **core Java fundamentals** such as classes, objects, encapsulation, collections, and user input handling.

---

## 📂 Project Structure
unit 1 project
│
├── LibraryManagementSystem.java
├── OnlineShopping.java
└── StudentManagement.java

---

## 📌 Programs Included

### 1️⃣ Library Management System
A console-based application to manage library operations.

**Features:**
- Add new books
- Display all books
- Issue books
- Return books
- Track availability status

**Concepts Used:**
- Classes & Objects
- Encapsulation
- ArrayList
- Conditional logic
- Scanner for user input

---

### 2️⃣ Online Shopping Cart System
A simple shopping cart simulation for an online store.

**Features:**
- View available products
- Add products to cart
- Increase product quantity
- View cart summary
- Calculate total price

**Concepts Used:**
- OOP principles
- HashMap
- Aggregation
- Menu-driven program

---

### 3️⃣ Student Management System
A console application to manage student records.

**Features:**
- Add student details
- Display all students
- Search student by ID

**Concepts Used:**
- Classes & methods
- ArrayList
- Data encapsulation
- Scanner input handling

---

## 🛠️ Technologies Used
- Java (Core Java)
- JDK 8 or above
- Command Line / Terminal
